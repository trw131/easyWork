# EasyWork - 工作日志记录系统

## 📋 项目简介

EasyWork 是一个基于 DevExpress WinForms 开发的工作日志记录与管理系统。该系统帮助用户快速记录每日工作内容,并提供月度报表导出、历史查询等功能,让工作记录变得简单高效。

## 🎯 核心功能

### 1. 今日日志记录
- **自动加载**: 启动软件自动加载当天日志,如已录入则显示现有内容
- **实时保存**: 内容变更时立即自动保存
- **定时备份**: 每30秒自动保存一次,防止数据丢失
- **保存提示**: 底部显示最后保存时间,让您放心记录

### 2. 快速复制功能
- **一键复制今日日志**: 点击"复制今日日志"按钮,自动将日期+内容复制到剪贴板
- **静默复制**: 无弹窗打扰,操作更流畅
- **格式化输出**: 复制内容包含日期标题,可直接粘贴使用

### 3. 历史日志查看
- **按月浏览**: 通过月份选择器快速切换不同月份
- **日期列表**: 左侧显示该月所有有记录的日期
- **内容预览**: 右侧实时显示选中日期的日志内容
- **双击复制**: 双击左侧日期列表,自动复制该日志到剪贴板
- **智能提示**: 复制后窗口标题显示"已复制"提示,2秒后自动恢复

### 4. 月度报表导出

#### 4.1 HTML格式导出
- **精美排版**: 绿色主题,卡片式布局,视觉清晰
- **响应式设计**: 适配不同屏幕尺寸
- **内嵌导出按钮**: HTML页面右上角提供"导出为TXT"按钮
- **浏览器内导出**: 直接在浏览器中点击按钮即可生成并下载TXT文件
- **统计信息**: 显示记录天数和导出时间

#### 4.2 TXT格式导出
- **简洁格式**: 纯文本格式,易于编辑和分享
- **清晰结构**: 包含标题、日期分隔、内容和统计信息
- **UTF-8编码**: 支持中文正确显示
- **两种导出方式**:
  - 直接导出为TXT文件
  - 从HTML页面中二次导出为TXT

### HTML格式特点
- 绿色主题,专业美观
- 卡片式布局,层次清晰
- 日期标题加粗显示
- 内容区域保留换行和空格
- 右上角嵌入"导出为TXT"功能按钮
- 底部显示统计信息

## 🚀 快速开始

### 安装步骤
1. 确保系统已安装 .NET Framework 4.8
2. 安装 DevExpress 控件库
3. 克隆或下载项目源代码
4. 使用 Visual Studio 打开解决方案
5. 还原 NuGet 包
6. 生成并运行项目

### 首次运行
1. 运行 `easyWork.exe`
2. 系统自动创建数据存储目录
3. 开始记录今日工作日志

3. **格式保留**: 复制的内容保留所有换行和格式

## 🔐 数据安全

### 自动保存机制
- **实时保存**: 每次内容变更立即保存
- **定时备份**: 每30秒自动保存一次
- **关闭保存**: 程序关闭前执行最后保存

### 数据备份建议
1. 定期导出月度报表作为备份
2. 将 `%AppData%\EasyWork\Logs\` 目录加入云同步
3. 重要内容建议额外保存副本

## 🐛 常见问题

### Q: 保存时间一直显示"尚未保存"?
A: 请确保输入了内容,空白日志不会触发保存。

### Q: 找不到历史记录?
A: 检查日期选择器是否选择了正确的月份。

### Q: 导出的HTML中文乱码?
A: 确保使用UTF-8编码的浏览器打开,现代浏览器一般自动识别。

### Q: 复制功能没反应?
A: 检查日志内容是否为空,空内容不会执行复制操作。

### Q: 可以修改历史日志吗?
A: 历史查看界面只能查看,如需修改请手动选择日期后在主界面编辑。
